# Astro bot : a discord password manager

## How to run

1. Change directory to `mate/app/` :  

```bash
cd mate/app/
```

2. Copy `example.env` to `.env` :  

```bash
cp example.env .env
```

3. Put your discord token in `.env`

4. Install docker and docker-compose

5. Run `docker-compose up --build`
